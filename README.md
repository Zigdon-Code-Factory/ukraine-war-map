# Russia-Ukraine War — Interactive Conflict Map

Interactive timeline and map visualization of the Russia-Ukraine war, backfilled for roughly the last year using the same event-driven format as the Iran war map project.

## Features

- Timeline playback
- Interactive Leaflet map
- Filtered event streams
- Cumulative damage tracking by target/location
- Source links on every event

## Data

- `data/events.json` is the primary event dataset
- Schema matches the existing Iran war map structure closely so the frontend can stay nearly identical

## Notes

This version is a backfilled starting point intended to mirror the original project shape for the Russia-Ukraine conflict.
